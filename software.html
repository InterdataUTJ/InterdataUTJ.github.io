Solución de Software

Para poder satisfacer las necesidades de nuestro cliente es indispensable el desarrollo de un sistema de software que permita manejar eficientemente el inventario de la farmacia, así como las ventas y la entrada y salida de ítems. Para poder lograr este cometido, Rocket Code desarrollo una aplicación en Java 17 que puede ser ejecutada por consola y le brinda al usuario una serie de herramientas que le permitirán a nuestro cliente desempeñar sus labores de una forma más eficientemente.

A continuación proporcionamos dos enlaces de descarga mediante los cuales es posible obtener la solución de software que desarrollamos, el primer enlace es el archivo JAR del proyecto, el cual puede ser ejecutado desde la consola usando el siguiente comando.

El segundo enlace proporciona un archivo comprimido RAR en el cual se encuentra el proyecto Java 17 que alberga el código que desarrollamos para nuestro cliente. Este proyecto utiliza la herramienta de compilación Ant, y fue desarrollado utilizando el IDE Apache Netbeans 17.

Para ejecutar tanto el proyecto como el archivo JAR es necesario contar con la versión 17 de Java o superior instalada.
